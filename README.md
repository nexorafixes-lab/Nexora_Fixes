# Nexora Fixes

Nexora Fixes is a digital marketing agency website foundation focused on business growth and customer acquisition. The copy centers on outcomes: turning clicks into customers through Meta Ads, Facebook Ads, conversion-focused creatives, web development, custom web solutions, Shopify e-commerce stores, and WordPress builds.

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Start with `app/page.tsx`, `componnets/Home/Hero.tsx`, and `app/lib/config.ts` when adjusting the homepage and SEO metadata.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to optimize DM Sans for body text and Outfit for headings.

## Deploy on Vercel

Deploy the Nexora Fixes site with the [Vercel Platform](https://vercel.com/new) or any host that supports Next.js.

Check out our [Next.js deployment documentation](https://nextjs.org/docs/app/building-your-application/deploying) for more details.
